# Portfolio Website

A premium, modern, and minimal single-page personal portfolio website built with HTML5, CSS3, and vanilla JavaScript. 

## Features
- **Pure Vanilla Stack**: No frameworks, libraries, or build tools. Just clean HTML, CSS, and JS.
- **Modern UI/UX**: Features glassmorphism, smooth scrolling, entrance animations, and a dark-themed professional color palette.
- **Fully Responsive**: Mobile-first design that looks great on all devices.
- **Performance Optimized**: Fast loading, lightweight, and SEO-friendly.
- **GitHub Pages Ready**: Can be deployed directly to GitHub pages without configuration.

## Deployment (GitHub Pages)
1. Initialize a git repository in this folder.
2. Commit all files.
3. Push to a new GitHub repository.
4. Go to repository Settings > Pages.
5. Select the `main` branch (or `master`) as the source and save.
6. Your portfolio will be live at `https://<username>.github.io/<repository-name>/`.
